# Missão Rebeca ✦

Uma aventura romântica em formato PWA, criada por Rafael especialmente para Rebeca.

O jogo mistura pixel art, exploração, mini-missões, planetas misteriosos, constelações, humor bobo na medida certa e um final com caixinha de aliança para transformar o pedido de namoro em uma surpresa interativa.

## Tecnologia

- React para HUD, botões e estados de interface
- Lucide React para os ícones
- Motion para microanimações
- Matter.js para física e colisões
- canvas-confetti para celebração
- Canvas 2D autoral para o motor de pixel art
- Vite para build rápido
- PWA com manifesto, ícones e service worker

## O jogo

- Rebeca controla uma exploradora em pixel art, com caminhada em 8 direções
- Controles por teclado, toque no mapa e joystick no celular
- Pingo, o guia emocional flutuante, acompanha a missão
- Sete órbitas/planetas com objetivos diferentes
- Selos orbitais para recuperar durante a aventura
- Mapa galáctico para revisitar mundos desbloqueados
- Diário secreto com bilhetes escondidos
- Glifos secretos que liberam uma frequência especial
- Puzzle de constelação com o nome Rebeca
- Minigame de voo espacial com meteoros e faíscas de coragem
- Trilha sonora chiptune procedural com botão de som
- Final cinematográfico com Rafael, Rebeca, caixinha de aliança e pedido de namoro

## Instalar no iPhone

1. Abra o link do app no Safari.
2. Toque no botão de compartilhar.
3. Escolha “Adicionar à Tela de Início”.
4. Abra o ícone “Missão Rebeca” como se fosse um app.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publicação

O projeto está configurado para Vercel usando `vercel.json`, com build em `npm run build` e saída em `dist`.
